HTML

HTML refers to Hypertext Markup Language, and it is the basic language used for web pages.

The head tag is used to include general data about the web page (such as the title), or things such as the page style, links, etc.

The body tag is where the structure of the page is done. Things like headings, paragraphs, boxes, divs and any other HTML tags and utilities are put inside the body tag.

The section tag is simply used to state a section in the document, by default it uses a block display.

The article tag is used to create separate content that is usually used for information that would be found on forums, blogs, etc.

The aside tag is used to write content separated from the content that it is placed in. Often styled as a sidebar.

A header tag is used to place introductory content, information or used for navigation.

A footer tag is, certainly, a tag used to place a footer on the web page. It can contain relevant information about the contents of the page, or about its authors.

CSS

CSS refers to Cascade Style Sheets, and it is used to style a web page along HTML.

There are three ways to use CSS with a HTML document. The first way is by internal CSS, using the style tag in the HTML document. The second way is by using the inline style attribute in HTML tags. The third way is by using an external CSS document and linking it in the HTML document.

A CSS selector is what is used to choose the HTML element that will be styled. It is probably the most important part of styling an HTML document. A CSS rule is a set of style properties that are applied to one or more HTML elements.

A tag selector is used to select all elements of a specific selector, or elements attached to a specific identifier such as a class or id.

A class selector is used to style all elements with a specific class identifier.

A class selector combined with a tag selector would select all the elements that are of the tag type that are inside the specified class.

An id selector is used to style tags that are identified by a specific id name.

The main difference between a class and an id is that classes are usually used for multiple tags and it is useful to use the same type of style for all elements from that class. An id is generally used more specifically for individual elements, which can be a single element or elements inside other tags.

A CSS box model represents the design and layout of a HTML element. It is a box that surrounds the element, which consists of margins, borders, padding and the content inside the box. All these elements can be styled with CSS.

You can easily change the font color of a text element by selecting the element and using the color attribute. It works for any text-type HTML element.
